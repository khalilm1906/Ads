DoubleClick DART Click Tracking

JavaScript Code

Original Code

You should have received some JavaScript code from DoubleClick DART that looks like the following: 


<SCRIPT language='JavaScript1.1' SRC="http://ad.doubleclick.net/adj/N270.site.com/B17804;sz=468x60;ord=[timestamp]?"></script>
<NOSCRIPT>
<A HREF="http://ad.doubleclick.net/jump/N270.site.com/B17804; sz=468x60;ord=[timestamp]?">
<IMG SRC="http://ad.doubleclick.net/ad/N270.site.com/B17804;sz=468x60;ord=[timestamp]?" BORDER=0 WIDTH=468 HEIGHT=60 ALT="Click Here!">Click Here!</A>
</NOSCRIPT>

Modified Code

What you need to do is modify the URL in the <SCRIPT> and <A> tags as shown by the highlighted changes: 


<SCRIPT language='JavaScript1.1' SRC="http://ad.doubleclick.net/adj/N270.site.com/B17804;sz=468x60;click=%%CLICKLINK%%;ord=[timestamp]?"></script>
<NOSCRIPT>
<A HREF="%%CLICKLINK%%http://ad.doubleclick.net/jump/N270.site.com/B17804; sz=468x60;ord=[timestamp]?">
<IMG SRC="http://ad.doubleclick.net/ad/N270.site.com/B17804;sz=468x60;ord=[timestamp]?" BORDER=0 WIDTH=468 HEIGHT=60 ALT="Click Here!">Click Here!</A>
</NOSCRIPT>

Warning

Make sure to preview the media and test clicking on it after it's created. If it fails to click through properly, try changing the bold %%CLICKLINK%% in the <SCRIPT> tag to %%CLICKTAG%% instead. This is sometimes necessary if DoubleClick themselves are sourcing the ad from another third party.

Note

The ad server recognizes the [timestamp] placeholder and will automatically replace it with a timestamp. It is not necessary to convert it to the %%TIMESTAMP%% macro manually.

IFRAME

Original Code

You should have received some IFRAME code from DoubleClick DART that looks like the following: 


<IFRAME SRC="http://ad.doubleclick.net/adi/N884.sitename/B3453509.2;sz=300x250;ord=[timestamp]?" WIDTH=300 HEIGHT=250 MARGINWIDTH=0 MARGINHEIGHT=0 HSPACE=0 VSPACE=0 FRAMEBORDER=0 SCROLLING=no BORDERCOLOR='#000000'>
<SCRIPT language='JavaScript1.1' SRC="http://ad.doubleclick.net/adj/N884.sitename/B3453509.2;abr=!ie;sz=300x250;ord=[timestamp]?">
</SCRIPT>
<NOSCRIPT>
<A HREF="http://ad.doubleclick.net/jump/N884.sitename/B3453509.2;abr=!ie4;abr=!ie5;sz=300x250;ord=[timestamp]?">
<IMG SRC="http://ad.doubleclick.net/ad/N884.sitename/B3453509.2;abr=!ie4;abr=!ie5;sz=300x250;ord=[timestamp]?" BORDER=0 WIDTH=300 HEIGHT=250 ALT="Click Here">
</A>
</NOSCRIPT>
</IFRAME>

Modified Code

What you need to do is modify the URL in the <IFRAME>, <SCRIPT> and <A> tags as shown by the highlighted changes: 


<IFRAME SRC="http://ad.doubleclick.net/adi/N884.sitename/B3453509.2;sz=300x250;click=%%CLICKLINK%%;ord=[timestamp]?" WIDTH=300 HEIGHT=250 MARGINWIDTH=0 MARGINHEIGHT=0 HSPACE=0 VSPACE=0 FRAMEBORDER=0 SCROLLING=no BORDERCOLOR='#000000'>
<SCRIPT language='JavaScript1.1' SRC="http://ad.doubleclick.net/adj/N884.sitename/B3453509.2;abr=!ie;sz=300x250;click=%%CLICKLINK%%;ord=[timestamp]?">
</SCRIPT>
<NOSCRIPT>
<A HREF="%%CLICKLINK%%http://ad.doubleclick.net/jump/N884.sitename/B3453509.2;abr=!ie4;abr=!ie5;sz=300x250;ord=[timestamp]?">
<IMG SRC="http://ad.doubleclick.net/ad/N884.sitename/B3453509.2;abr=!ie4;abr=!ie5;sz=300x250;ord=[timestamp]?" BORDER=0 WIDTH=300 HEIGHT=250 ALT="Click Here">
</A>
</NOSCRIPT>
</IFRAME>

Warning

Make sure to preview the media and test clicking on it after it's created. If it fails to click through properly, try changing the bold %%CLICKLINK%% in the <SCRIPT> and <IFRAME> tags to %%CLICKTAG%% instead. This is sometimes necessary if DoubleClick themselves are sourcing the ad from another third party.

Note

The ad server recognizes the [timestamp] placeholder and will automatically replace it with a timestamp. It is not necessary to convert it to the %%TIMESTAMP%% macro manually.

IMG Code

Original Code

You should have received some IMG code from DoubleClick DART that looks like the following: 


<A HREF="http://ad.doubleclick.net/jump/N884.sitename/B2343505.2;sz=300x250;ord=[timestamp]?">
<IMG SRC="http://ad.doubleclick.net/ad/N884.sitename/B2343505.2;sz=300x250;ord=[timestamp]?" BORDER=0 WIDTH=300 HEIGHT=250 ALT="Click Here">
</A>

Modified Code

What you need to do is modify the URL in the <A> tag as shown by the highlighted changes: 


<A HREF="%%CLICKLINK%%http://ad.doubleclick.net/jump/N884.sitename/B2343505.2;sz=300x250;ord=[timestamp]?" target="_blank">
<IMG SRC="http://ad.doubleclick.net/ad/N884.sitename/B2343505.2;sz=300x250;ord=[timestamp]?" BORDER=0 WIDTH=300 HEIGHT=250 ALT="Click Here">
</A>

Note

The ad server recognizes the [timestamp] placeholder and will automatically replace it with a timestamp. It is not necessary to convert it to the %%TIMESTAMP%% macro manually.
